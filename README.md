# mobillity

Open course materials from **Mobillity** — practical, hands-on data analysis done with the help of an LLM, on public European mobility data.

Each top-level folder is one course.

## Courses
- [**mobillity-univ**](./mobillity-univ/) — Mobillity University. Start here: [open the materials](./mobillity-univ/index.md).

_More courses will be added as they are produced._

## How to use
Open any notebook's **Open in Colab** badge to run it with no setup, or browse a course's `index.md` for all its links. Notebooks generate their own small, synthetic datasets — no downloads, no real personal data.
