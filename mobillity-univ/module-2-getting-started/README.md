# Module 2 — Getting Started with LLM-Assisted Data Analysis

The LLM-assisted workflow, loading data, inspecting with pandas, what happens when data
formats go wrong, and structured data formats. Taught through the lecture videos; the
sample dataset here lets you try the loading/inspection steps yourself.

## Contents
- `bumpers/` — `epic-2-intro.html`, `epic-2-conclusion.html`.
- `data/barcelona-bicing/` — `sample.csv` (~9 MB) + `dataset-card.md` + `README-usage.md`.
- `data/GET-DATA.md` — how to fetch the full 353 MB snapshot if you want it.

## Try it
Load `data/barcelona-bicing/sample.csv` with pandas and run `.shape`, `.info()`,
`.describe()`. Watch for the European date format and the status codes the card documents.
